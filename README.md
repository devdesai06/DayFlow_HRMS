# DayFlow

## Project Overview
DayFlow is a time management application designed to enhance productivity by allowing users to track their tasks and manage their daily activities seamlessly. It provides a user-friendly interface to help users plan their day effectively, set reminders, and analyze their productivity patterns.

## Setup Instructions
To set up DayFlow locally, follow these steps:
1. **Clone the Repository:**  
   ```bash
   git clone https://github.com/OWNER/DayFlow.git
   cd DayFlow
   ```
2. **Install Dependencies:**  
   ```bash
   npm install
   ```
3. **Run the Application:**  
   ```bash
   npm start
   ```
4. **Access the Application:**  
   Open your browser and go to `http://localhost:3000`.

## Features
- **Task Management:** Create, update, and delete tasks at ease.
- **Calendar Integration:** Sync tasks with your calendar.
- **Reminders:** Set reminders for important tasks.
- **Analytics:** View productivity reports and insights.

## Tech Stack
DayFlow is built using the following technologies:
- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Deployment:** Frontend : Vercel Backend : Render

## Usage Guidelines
1. **Creating Tasks:**  
   Click on the "Add Task" button, fill in the task details, and save.
2. **Viewing Tasks:**  
   Navigate to the "Tasks" section to view all your tasks.
3. **Updating Tasks:**  
   Click on a task to edit its details and save changes.
4. **Deleting Tasks:**  
   Click on the delete icon next to a task to remove it.
